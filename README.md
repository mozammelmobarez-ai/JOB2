---

## 🚀 راه‌اندازی روی Railway (رایگان!)

### قدم ۱: ساخت GitHub Repository

1. برو به **github.com** و لاگین کن
2. بزن **New repository**
3. اسم بذار: `job-ai-bot`
4. فایل‌های این پوشه رو آپلود کن

### قدم ۲: Deploy روی Railway

1. برو به: **railway.app**
2. با GitHub لاگین کن
3. بزن **New Project** → **Deploy from GitHub**
4. ریپوی `job-ai-bot` رو انتخاب کن
5. تمام! 🎉

### قدم ۳: شروع استفاده

1. برو به ربات تلگرام
2. بزن `/start`
3. بیوگرافی‌تون رو بفرستید
4. کانال‌های کاریابی اضافه کنید
5. ربات هر 5 ساعت خودکار جستجو می‌کنه!
